# 项目简介

## 核心功能
- 基于YOLOv8的塔吊识别检测系统，支持图片、视频及摄像头进行检测
- 提供6000张标注数据集，包含['Tower-crane']类别
- 支持选择不同检测模型，可自定义置信度和IOU阈值
- 界面实时显示目标位置、目标总数和置信度信息
- 支持批量检测并查看所有检测结果
- 可保存检测结果，支持选择显示标签和原图
- 提供完整的UI界面源文件和图标

## 适用场景
- 建筑工地塔吊安全监控
- 工程施工现场管理
- 设备安全检测与识别
- 视频监控分析
- 批量图像处理

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 依赖python3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 已训练好模型，配置环境后可直接使用
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整的项目运行教程和源码文件
